# Downloading the Build Artifact from GitHub Actions

1. Open the GitHub repository.
2. Click **Actions** tab.
3. Open the workflow run named **Build Deployable ZIP**.
4. Verify all jobs are green.
5. Scroll to **Artifacts** section.
6. Download artifact named `datai-deployable-zip`.
7. Unzip locally and confirm it contains Laravel app files plus `public/assets/angular-v4/browser`.
